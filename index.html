Payment cards

Cards used by the agency to pay for tickets. Identifiers follow the format AX1017 (AmEx ending 1017), VI4242 (Visa ending 4242), CA5500 (MasterCard ending 5500). Used by the parser to auto-match payments from PNRs.

📥 Import GDS sales report

Drop or select a file from your GDS. Format is auto-detected:

  • Amadeus Link Sales Report (.csv) — daily sales from Amadeus Selling Platform
  • Amadeus Provider Link General Ticket Report (.csv) — extended ticket report with refund/void details
  • Galileo Agent Report XLS (.xls) — from golink.galileo.com.ua, includes card numbers & departure times (preferred)
  • Galileo Export CSV / XML — fallback if XLS unavailable
  • MIR / SPRK Lufthansa / Aegean (coming next phase)
